Мета роботи

Освоїти повний цикл інтеграції реляційних даних у середовищі R: від завантаження даних з різноманітних джерел (CSV, Excel, Google Таблиці) до їх об’єднання за допомогою вертикальних (bind_rows) та горизонтальних (left_join, semi_join, anti_join) операцій. Навчитися будувати реалістичні ETL-конвеєри, які моделюють типові задачі інженера даних у production-середовищі.

Хід роботи

Виконати індивідуальне завдання варіанту 3:

  1. Згенерувати датасети:
    • ga_november.csv та ga_december.csv — щоденні звіти Google Analytics за листопад і грудень.
      Стовпці: date, page, sessions, users, bounce_rate, avg_session_duration.
    • content_plan.xlsx з аркушами:
      • pages — довідник сторінок (page_url, page_title, category, author)
      • campaigns_nov — маркетингові кампанії за листопад (page_url, campaign, channel)
      • campaigns_dec — аналогічно для грудня.
  2. Завантаження даних
    • Завантажити обидва CSV-файли за допомогою vroom.
    • Завантажити всі аркуші з content_plan.xlsx за допомогою readxl.
  3. Вертикальне об’єднання
    • Об’єднати ga_november.csv і ga_december.csv у єдину таблицю ga_full.
    • Додати стовпець month зі значеннями "2023-11" та "2023-12".
  4. Горизонтальне об’єднання
    • Додати до ga_full інформацію про сторінки (page_title, category, author) через left_join за ключем page = page_url.
    • Об’єднати таблиці кампаній (campaigns_nov, campaigns_dec) у all_campaigns, додавши стовпець month.
    • Додати дані про кампанії (campaign, channel) до ga_full через left_join за page і month.
  5. Агрегація та аналіз
    • Для кожної категорії сторінок і місяця обчислити: загальну кількість сесій, середній час сесії, середній bounce rate.
    • Визначити топ-5 сторінок за кількістю користувачів у кожному місяці.
  6. Фільтруюче з’єднання
    • Визначити сторінки, які мали маркетингову кампанію в обох місяцях (semi_join).
    • Визначити сторінки, які були в кампаніях у листопаді, але не в грудні (anti_join).

generate_ga <- function(start_date, end_date, pages_vec) {
  dates <- seq.Date(as.Date(start_date), as.Date(end_date), by = "day")
  expand_grid(date = dates, page = pages_vec) %>%
    mutate(
      sessions             = pmax(1, round(rnorm(n(), mean = 45, sd = 20))),
      users                = pmax(1, round(sessions * runif(n(), 0.6, 0.9))),
      bounce_rate          = round(runif(n(), 0.25, 0.75), 2),
      avg_session_duration = round(runif(n(), 60, 420))
    )
}

ga_nov <- generate_ga("2023-11-01", "2023-11-30", pages$page_url)
ga_dec <- generate_ga("2023-12-01", "2023-12-31", pages$page_url)

5. Фільтруюче з’єднання

5.1 Сторінки з кампаніями в обох місяцях (semi_join)
Code 
camp_nov_df <- all_campaigns %>% filter(month == "2023-11")
camp_dec_df <- all_campaigns %>% filter(month == "2023-12")

both_months <- camp_nov_df %>%
  semi_join(camp_dec_df, by = "page_url") %>%
  distinct(page_url) %>%
  left_join(pages_ref %>% select(page_url, page_title, category), by = "page_url")
